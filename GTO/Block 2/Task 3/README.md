# Задача 3
Проверьте, является ли число простым.
## Входные данные:
Вводится одно натуральное число n, принимающие значения от 2 до 2*10^5.
## Пример ввода:
1) 3

2) 4
## Выходные данные:
Необходимо вывести "prime", если число простое, или "composite", если число составное.
## Пример вывода:
1) prime

2) composite
